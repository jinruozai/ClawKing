# 龙虾之王 - 属性系统说明

## 基础属性

### 核心属性（玩家可分配，总点数20点）
1. **health_points**（生命值）：
   - 每点增加龙虾的最大生命值
   - 影响龙虾的生存能力和耐久度

2. **magic_points**（魔法值）：
   - 每点增加龙虾的最大魔法值
   - 影响技能使用频率和持续能力

3. **attack**（攻击力）：
   - 每点增加龙虾的基础攻击力
   - 影响物理伤害输出

4. **defense**（防御力）：
   - 每点减少受到的物理伤害
   - 提高生存能力

5. **movement_speed**（移动速度）：
   - 每点增加每回合的最大移动距离
   - 影响龙虾的机动性和战术位置选择

6. **backpack_size**（背包大小）：
   - 每点增加可携带物品的数量
   - 影响战术选择和物品使用策略

### 扩展属性
1. **attack_range**（攻击范围）：
   - 每点增加攻击可以达到的格子数
   - 远程武器可以增加攻击范围，但可能会降低攻击力

2. **combo_chance**（连击率）：
   - 每点增加10%的连击几率
   - 触发时会追加一次攻击

3. **critical_chance**（暴击率）：
   - 每点增加10%的暴击几率
   - 触发时会造成额外伤害

4. **attack_count**（连击次数）：
   - 每点增加最大连击次数
   - 影响多段攻击的潜力

5. **critical_multiplier**（暴击倍率）：
   - 每点增加10%的暴击伤害倍数
   - 影响暴击时的伤害输出

### 元素属性
1. **fire_damage**（火焰伤害）：
   - 每点增加火焰系技能和武器的伤害
   - 关联灼烧效果的伤害

2. **ice_damage**（冰系伤害）：
   - 每点增加冰系技能和武器的伤害
   - 关联冰冻和减速效果

3. **lightning_damage**（雷系伤害）：
   - 每点增加雷系技能和武器的伤害
   - 关联麻痹效果

4. **water_damage**（水系伤害）：
   - 每点增加水系技能和武器的伤害
   - 关联治疗和减速效果

5. **poison_damage**（毒系伤害）：
   - 每点增加毒系技能和武器的伤害
   - 关联持续伤害效果

6. **wood_damage**（木系伤害）：
   - 每点增加木系技能和武器的伤害
   - 关联治疗和生命恢复效果

## 特殊属性

### 战斗相关
1. **attack_speed**（攻击速度）：
   - 每点增加10%的攻击速度
   - 影响攻击频率

2. **evasion**（闪避率）：
   - 每点增加10%的闪避几率
   - 触发时可以完全躲避攻击

3. **lifesteal**（生命汲取）：
   - 每点增加10%的生命汲取效果
   - 攻击时恢复生命值

4. **armor_penetration**（破甲）：
   - 每点增加10%的破甲效果
   - 无视目标部分防御

5. **damage_reflection**（伤害反射）：
   - 每点增加10%的伤害反射效果
   - 受到攻击时反弹部分伤害

6. **invulnerability**（无敌）：
   - 每点提供一次无敌效果
   - 免疫一次攻击

7. **resurrection_chance**（复活几率）：
   - 每点增加10%的复活几率
   - 死亡后有几率复活

### 控制效果
1. **paralysis_chance**（麻痹几率）：
   - 每点增加10%的麻痹几率
   - 使目标无法行动

2. **silence_chance**（沉默几率）：
   - 每点增加10%的沉默几率
   - 使目标无法使用技能

3. **movement_speed_reduction**（移动速度减少）：
   - 每点减少目标10%的移动速度
   - 影响目标的机动性

4. **attack_reduction**（攻击力减少）：
   - 每点减少目标10%的攻击力
   - 削弱目标的伤害输出

### 其他属性
1. **healing**（治疗效果）：
   - 每点增加治疗效果的数值
   - 影响生命恢复量

2. **magic_cost**（魔法消耗）：
   - 每点增加技能的魔法消耗
   - 影响技能使用的魔法成本

3. **action_count**（行动次数）：
   - 每点增加每回合的行动次数
   - 影响每回合可以执行的操作数

4. **duration**（持续时间）：
   - 每点增加效果的持续回合数
   - 影响 buff/debuff 的持续时间

## 属性计算规则

### 百分比属性计算
- 所有百分比相关的属性，每个点数代表10%
- 例如：1点 critical_chance 代表10%的暴击几率

### 元素伤害计算
- 元素伤害 = 基础攻击力 × (1 + 对应元素属性 × 0.1) × 目标元素抗性修正
- 例如：基础攻击力10，火属性3点，目标无火抗性，则火焰伤害 = 10 × (1 + 3 × 0.1) = 13

### 伤害计算
- 物理伤害 = (攻击力 - 目标防御力) × 各种修正系数
- 元素伤害 = 基础攻击力 × 元素属性修正 × 目标抗性修正
- 最终伤害 = 物理伤害 + 元素伤害

### 连击与暴击
- 连击：触发后追加一次攻击，伤害为原伤害的80%
- 暴击：造成150% + 暴击倍率的伤害

## 装备与技能属性修正
- 装备和技能通过 attribute_modifiers 数组来修改属性
- 每个修改项为 [属性名, 数值] 格式
- 装备和技能可以同时提供增益和减益效果
- 例如：范围斧 [["attack",15],["attack_range",1],["attack",-3]] 表示增加15点攻击力和1点攻击范围，但减少3点攻击力

## 流派建议

### 近战输出流
- 高攻击、高连击率、高暴击率
- 适合使用高伤害武器和连击技能

### 远程消耗流
- 高攻击范围、高元素伤害、高魔法值
- 适合使用远程武器和元素技能

### 防御坦克流
- 高防御、高生命值、高伤害反射
- 适合使用防御装备和护盾技能

### 速度机动流
- 高移动速度、高闪避率、高攻击速度
- 适合使用速度装备和控制技能

### 魔法辅助流
- 高魔法值、高治疗效果、高控制效果
- 适合使用辅助技能和治疗装备
